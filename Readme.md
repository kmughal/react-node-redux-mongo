# Mongodb on mac

- Run following command :
cmd`
brew services start mongodb
`
- To stop the mongo server:
cmd`
brew services stop mongodb
`

- To restart mongo server:
cmd`
brew services restart mongodb
`

- To get mongo db informatiion:
cmd `
brew info mongodb
`
